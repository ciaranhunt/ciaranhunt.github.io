3.4 日期选择

html

在script标签内配置：

JavaScript

基础参数选项：

  • type：控件类型-year、month、date、time、datetime

  • range：选择范围-true/false/～来自定义分隔符

  • format：

  • value：初始值-指定时间(‘2020-2-2’)/new Date()

  • min/max:最小/大范围内的日期时间值-具体时间(‘2015-02-23’具体格式与type一致)/±7(过去7天、将来7天)

  • showBottom：是否显示底部栏-true/false

  • btns：工具按钮-[‘clear’, ‘now’, ‘confirm’]

  • theme：主题-default（默认简约）、molv（墨绿背景）、#颜色值（自定义颜色背景）、grid（格子主题）

基础回调函数

  • 控件初始打开的回调：ready: function(date){…}
  • 日期时间被切换后的回调：change: function(value, date, endDate){…}（生成的值、日期时间对象、结束的日期时间对象）
  • 控件选择完毕后的回调：done: function(value, date, endDate){…}（）（生成的值、日期时间对象、结束的日期时间对象）

3.5 表单

必须给表单体系所在的父元素加上class="layui-form"，一切的工作都会在你加载完form模块后，自动完成。

预设的元素属性，他们可以使得一些交互操作交由form模块内部、或者你来借助form提供的JS接口精确控制：

  • 下拉框lay-search属性；可以搜索

重新渲染的：(需要引入form模块)

  • 下拉列表
  • 单选框
  • 复选框
  • 开关

富文本需要引入富文本模块并且初始化。

4. 表单对象

  • 监听事件语法：form.on(‘event(过滤器值)’, callback);

    • 监听提交事件
    • 监听下拉框改变事件
    • 监听复选框改变事件
    • 监听开关改变事件
    • 监听单选框改变事件

  • 表单赋值/取值
    语法：form.val(‘filter’, object);

  • 表单验证
    大多数时候只需要在表单元素上加上lay-verify=""属性值即可

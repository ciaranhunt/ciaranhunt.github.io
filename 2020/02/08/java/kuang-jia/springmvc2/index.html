5. 自定义拦截器

拦截器与过滤器比较相像的技术，发送请求时被拦截器拦截，在控制器的前后添加额外的功能。

与AOP不同，AOP是拦截方法，针对服务层的实现类；而拦截器是请求的拦截，针对控制器的方法。
与filter不同，filter拦截任何请求；而拦截器只能拦截控制器的。

  1. 实现HandlerInterceptor接口。

    • preHandle：在进入控制器之前执行，如果返回false，阻止进入控制器。(状态检查)
    • postHandle：在控制器完成，进入jsp之前执行。(日志功能，敏感词过滤)
    • afterCompletion：jsp执行完成后执行(记录异常)

  2. 在springmvc.xml配置

    • 直接配置bean标签，拦截所有请求

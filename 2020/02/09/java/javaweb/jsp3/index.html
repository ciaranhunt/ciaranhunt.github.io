1. 九大内置对象

名称 作用 获取方式
request 封装所有请求信息 方法参数
response 封装所有响应信息 方法参数
session 封装所有会话信息 req.getSession()
application 所有信息 getServletContext()/req.getServletContext()
out 输出对象 response.getWriter()
exception 异常对象
page 当前页面对象
pageContext 获取其它对象
config 配置信息

2. 四大作用域

2.1 request

在一次请求中是同一个对象，下次请求重新实例化

2.2 page

当前页面不会重新实例化

2.3 session

一次会话中，只要客户端的sessionid不变，对象就不变(不超过默认时间)

2.4 application

只有器启动服务器才实例化对象

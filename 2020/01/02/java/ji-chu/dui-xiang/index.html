4. 字段

4.4.1 字段介绍

如果某类的某个成员会是基本数据类型，JAVA会确保它获得一个默认值。方法的局部变量不会确保初始化。

4.4.2 static关键字

使用情形：

  1. 只想为某特定域分配单一存储空间，而不用去考虑究竟要创建多少对象，甚至根本不用创建对象。（针对类字段）
  2. 即使没有创建对象，也能调用这个方法。（针对类方法）
